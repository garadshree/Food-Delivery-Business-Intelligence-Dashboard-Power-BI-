# Food-Delivery-Business-Intelligence-Dashboard-Power-BI-
End-to-end Power BI project analyzing food delivery data to uncover insights on customer behavior, order trends, restaurant performance, and payment preferences. Built using star schema modeling, DAX, and interactive dashboards for business decision-making.

# 🍽️ Food Delivery Business Intelligence Dashboard (Power BI)

## 🔹 Project Overview

This project focuses on analyzing food delivery business data to generate actionable insights and support decision-making. It transforms raw operational data into an interactive Power BI dashboard.

## 🔹 Business Problem

The objective was to provide visibility into order trends, customer segments, delivery performance, and payment preferences.

## 🔹 Data Sources

* Customer Details
* Food Details
* Restaurant Details
* Sales / Transaction Data

## 🔹 Data Modeling

Implemented a **Star Schema**:

* Fact Table: Sales Transactions
* Dimension Tables: Customer, Food, Restaurant

## 🔹 Dashboard Features

* KPI Cards (Total Quantity, Avg Quantity, Transactions)
* Monthly trend analysis
* Food item performance
* Delivery vs cancellation analysis
* Payment insights
* Member type contribution

## 🔹 Key Insights

* Majority orders successfully delivered
* UPI is the most preferred payment method
* Gold members contribute higher transactions
* Seasonal trends in order volume

## 🔹 Tools Used

* Power BI
* DAX
* Data Modeling
* Data Visualization
